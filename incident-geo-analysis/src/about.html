About

This project displays police incidents in Garland, TX on a map. Some addresses are marked confidential and cannot be plotted. Those can be viewed by expanding the "Confidential Addresses" section in the bottom right.

It grabs data from PDF files uploaded to: Garland, TX Crime Statistics & Maps

You can find out more about the specifics here: source code

Roadmap

  • Persist all these incidents in a db
  • Once incidents are persisted this will allow us to easily query and analyze the data
  • Allow looking a more historical data, previous weeks
  • Highlighting certain incidents
  • Automating the process that gets the data
